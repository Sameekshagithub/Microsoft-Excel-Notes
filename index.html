Introduction to Excel

What is Microsoft Excel?

Microsoft Excel is a powerful spreadsheet application developed by Microsoft. It is used for organizing, analyzing, and visualizing data through a grid of rows and columns. Excel is part of the Microsoft Office / Microsoft 365 suite and is widely used in business, finance, education, and data analysis.

Data Organization

Store and manage data in structured rows and columns with unlimited flexibility.

Calculation

Perform complex mathematical, statistical, financial, and logical calculations.

Visualization

Create charts, graphs, and dashboards to present data visually and clearly.

Automation

Automate repetitive tasks using macros and VBA scripting.

Key Terminology

TermDefinition
WorkbookThe entire Excel file (.xlsx) containing one or more sheets
WorksheetA single spreadsheet tab within a workbook
CellThe intersection of a row and a column (e.g., A1, B5)
RowHorizontal line of cells numbered 1, 2, 3…
ColumnVertical line of cells labeled A, B, C…
RangeA group of selected cells (e.g., A1:D10)
FormulaAn equation that performs calculations, starting with =
FunctionA built-in predefined formula (e.g., SUM, AVERAGE)

Interface & Navigation

Excel Interface Components

  • Title Bar – Shows the file name and application name at the very top
  • Ribbon – The toolbar area with tabs: Home, Insert, Page Layout, Formulas, Data, Review, View
  • Name Box – Displays the address of the selected cell (e.g., A1)
  • Formula Bar – Shows the content or formula of the active cell
  • Column Headers – Letters A, B, C… at the top identifying columns
  • Row Headers – Numbers 1, 2, 3… on the left identifying rows
  • Sheet Tabs – Tabs at the bottom for navigating between worksheets
  • Status Bar – At the bottom; shows sum, count, average of selected cells
  • Scroll Bars – Used to navigate large spreadsheets horizontally and vertically

Cell Addressing

TypeExampleDescription
Relative=A1+B1Changes when formula is copied to another cell
Absolute=$A$1+$B$1Does not change when formula is copied; fixed reference
Mixed=$A1 or =A$1One part fixed, one part relative
Tip: Press F4 to toggle between reference types while editing a formula.

Data Entry & Formatting

Types of Data in Excel

  • Text (Labels) – Alphabetic or alphanumeric data. Aligns left by default.
  • Numbers (Values) – Numeric data used in calculations. Aligns right by default.
  • Dates & Times – Stored as numbers internally; displayed as dates.
  • Logical (Boolean) – TRUE or FALSE values used in conditions.
  • Error Values – #DIV/0!, #VALUE!, #REF!, #NAME?, #N/A, #NULL!, #NUM!

Formatting Options

Number Formats

General, Number, Currency, Accounting, Date, Time, Percentage, Fraction, Scientific, Text

Cell Alignment

Left, Center, Right, Top, Middle, Bottom, Wrap Text, Merge Cells

Font Styling

Bold, Italic, Underline, Font Color, Font Size, Font Family

Borders & Fills

All Borders, Thick Box Border, Cell Background Color, Pattern Style

Conditional Formatting

Automatically applies formatting (color, icons, data bars) based on cell values or rules.

  • Highlight Cell Rules – Greater Than, Less Than, Equal To, Duplicate Values
  • Top/Bottom Rules – Top 10 Items, Bottom 10%, Above/Below Average
  • Data Bars – Visual bar inside the cell proportional to the value
  • Color Scales – Gradient color based on value range (e.g., Red–Yellow–Green)
  • Icon Sets – Arrows, traffic lights, stars based on thresholds
  • New Rule – Write custom formula-based rules
Tip: Use Home → Conditional Formatting → Manage Rules to edit or delete existing rules.

Formula Basics

All formulas in Excel begin with an equals sign (=). Formulas can include operators, cell references, values, and functions.

Arithmetic Operators: + (Add)  |  − (Subtract)  |  * (Multiply)  |  / (Divide)  |  ^ (Exponent)  |  % (Percent)

Logical & Lookup Functions

Logical Functions

FunctionSyntaxDescription
IF=IF(condition, value_if_true, value_if_false)Performs conditional test

Charts & Visualizations

Types of Charts

Column / Bar Chart

Comparing categories. Best for discrete data sets side by side.

Line Chart

Showing trends over time. Best for continuous data like sales over months.

Pie / Doughnut Chart

Showing proportions. Best for displaying parts of a whole (use <6 slices).

Area Chart

Like line chart but filled. Good for cumulative totals over time.

Scatter Plot

Showing correlations. Best for comparing two numeric variables.

Combo Chart

Combines bar and line. Useful for comparing data with different scales.

Histogram

Shows frequency distribution. Good for statistical analysis of data ranges.

Waterfall Chart

Shows cumulative effect of sequential values. Great for financial analysis.

Creating a Chart — Steps

  • Select the data range including headers
  • Go to Insert tab → Charts group
  • Click the desired chart type (or use Recommended Charts)
  • The chart is inserted; use Chart Design tab to customize style
  • Use Format tab to adjust colors, fonts, borders
  • Click Chart Elements (+) to add/remove title, legend, gridlines, data labels
  • Right-click chart → Select Data to modify data range
  • Move chart to a separate sheet using Move Chart

Sparklines

Sparklines are tiny charts inside a single cell that show data trends. They are useful for dashboards and compact reports.

  • Line Sparkline – Shows trend direction over a data series
  • Column Sparkline – Shows relative values as bars in a cell
  • Win/Loss Sparkline – Shows positive/negative outcomes only
Insert Sparkline: Insert → Sparklines → Choose type → Select data range → Choose location cell

PivotTables

What is a PivotTable?

A PivotTable is an interactive tool that summarizes, analyzes, explores, and presents data. It allows you to rearrange (pivot) data dynamically without writing formulas. It's one of Excel's most powerful features for data analysis.

Creating a PivotTable

  • Click anywhere inside your data table
  • Go to Insert → PivotTable
  • Choose: New Worksheet or Existing Worksheet
  • The PivotTable Field List appears on the right
  • Drag fields into: Filters, Rows, Columns, Values areas
  • Use the PivotTable Analyze tab for advanced options

PivotTable Areas Explained

AreaPurposeExample
RowsGroup data vertically by categoriesProduct names
ColumnsGroup data horizontally by categoriesMonths or regions
ValuesAggregate data (Sum, Count, Average, etc.)Total Sales
FiltersFilter the entire table by a fieldYear, Department

PivotChart & Slicers

  • PivotChart – A chart connected to a PivotTable; updates automatically when data changes
  • Slicers – Visual filter buttons for PivotTables; click to filter by category
  • Timelines – Date-based slicers for filtering by month, quarter, year
  • Calculated Field – Add a custom formula-based field to the PivotTable
Refresh PivotTable: Right-click on the PivotTable → Refresh (or press Alt + F5) whenever source data changes.

Data Tools

Sorting & Filtering

  • Sort A→Z / Z→A – Sort text or numbers in ascending or descending order
  • Custom Sort – Sort by multiple levels (e.g., by Department, then by Name)
  • AutoFilter – Click dropdown arrows on column headers to filter rows
  • Advanced Filter – Filter with complex criteria using a criteria range
  • Clear Filter – Data → Clear to remove all active filters

Data Validation

Restricts what can be entered in a cell. Go to Data → Data Validation.

TypeUse Case
Whole NumberAllow only integers within a range
DecimalAllow decimals within specified limits
ListDropdown menu from a defined list or range
DateRestrict to dates within a range
Text LengthRestrict character count
CustomWrite your own formula as the validation rule

Remove Duplicates & Text to Columns

  • Remove Duplicates – Data → Remove Duplicates → select columns to check
  • Text to Columns – Split text in one column into multiple columns (by delimiter or fixed width)
  • Flash Fill – Automatically fills data based on a pattern you establish (Ctrl+E)
  • Consolidate – Combine data from multiple ranges into one

What-If Analysis Tools

  • Goal Seek – Find the input value needed to achieve a specific result (Data → What-If Analysis → Goal Seek)
  • Scenario Manager – Compare multiple scenarios with different input values
  • Data Table – Show how one or two variables affect a formula result
  • Solver – Advanced optimization tool (must enable via Add-ins)

Advanced Features

Named Ranges

Assign a name to a cell or range to use instead of cell addresses in formulas.

  • Select the range → click the Name Box → type a name → press Enter
  • Or go to Formulas → Define Name
  • Use in formulas: =SUM(SalesData) instead of =SUM(B2:B100)
  • Manage names via Formulas → Name Manager

Excel Tables (Structured References)

Convert a range to a formal Excel Table using Ctrl+T or Insert → Table.

  • Tables auto-expand when new data is added
  • Formulas use structured references: =SUM(Table1[Sales])
  • AutoFilter is automatically enabled on tables
  • Total Row can be added for quick aggregations
  • Tables make VLOOKUP/XLOOKUP ranges dynamic

Array Formulas & Dynamic Arrays (Excel 365)

FunctionDescription
=UNIQUE(A1:A20)Returns list of unique values from a range
=SORT(A1:A20)Sorts an array or range
=SORTBY(A:A,B:B)Sorts by another column's values
=FILTER(A:B,C:C="Yes")Returns rows matching a condition
=SEQUENCE(10,1,1,1)Generates a sequence of numbers
=RANDARRAY(5,3)Generates array of random numbers
Dynamic array functions spill results automatically into adjacent cells — no need for Ctrl+Shift+Enter. The spill range shows with a blue border.

Macros & VBA Basics

  • A Macro is a recorded set of actions to automate repetitive tasks
  • Record Macro: View → Macros → Record Macro → perform actions → Stop Recording
  • Run Macro: View → Macros → View Macros → Run
  • VBA Editor: Press Alt+F11 to open the Visual Basic for Applications editor
  • Save macro-enabled workbooks as .xlsm format
  • Assign macros to buttons for easy access from the sheet

Protecting Workbooks & Sheets

  • Protect Sheet – Review → Protect Sheet → set password and permissions
  • Protect Workbook – Prevents adding/deleting sheets or changing structure
  • Lock/Unlock Cells – Format Cells → Protection → uncheck Locked for editable cells
  • Encrypt Workbook – File → Info → Protect Workbook → Encrypt with Password

Power Query (Get & Transform)

Power Query is Excel's data import and transformation engine (Data → Get Data).

  • Import data from Excel, CSV, Web, SQL, SharePoint, and 100+ sources
  • Clean data: Remove nulls, split columns, change data types
  • Merge and append multiple tables or files
  • All steps are recorded and can be refreshed with one click
  • No formula writing needed — GUI-based transformation
